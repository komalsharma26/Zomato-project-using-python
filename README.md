📌 Overview

This project performs Exploratory Data Analysis (EDA) on the Zomato dataset using Python. It helps in understanding restaurant trends, customer preferences, ratings, pricing, and location-based insights.

🚀 Features
Data cleaning and preprocessing
Analysis of restaurant ratings and reviews
Cost distribution and pricing insights
Location-based restaurant analysis
Data visualization using graphs and charts
🛠️ Tech Stack
Python
Pandas
NumPy
Matplotlib / Seaborn
📊 Key Insights
Identified top-rated restaurants and cuisines
Analyzed price range vs ratings
Explored city-wise restaurant distribution
Found customer preference patterns
📁 Dataset
Zomato dataset (CSV format)
▶️ How to Run
Clone the repository
git clone https://github.com/your-username/zomato-project.git
Install required libraries
pip install pandas numpy matplotlib seaborn
Run the notebook/script
jupyter notebook

This project provides meaningful insights into the food industry and helps understand customer behavior using data analysis techniques.
